Write the functions described:


1. Write a function, sumNums, that takes in two numbers as input, num1 and num2, and returns their sum.

2. Write a function, multNums, that takes in two numbers as input, num1 and num2, and returns their product.

3. Write a function, sumThree, that takes in three numbers as input, n1, n2 and n3, and returns their sum.

4. Write a function, largestNum, that takes in three numbers as input, b1, b2 and b3, and returns the largest one.  If all numbers are the same, return -1.

5. Write a function, greetPerson, that takes in a person's name, firstName, and returns a greeting that includes the name.

6. Write a function, numCookies, that takes in a number of cookies, c, and returns a statement telling us how many cookies we have.

7. Write a function, isGreaterThan100, that takes in a number, myNumber, and returns true if the number is greater than 100 and false otherwise. 

8. Write a function, equalStrings, that takes in two strings, s1 and s2, and returns true if the strings are the same, false if they are not.

9. Write a function, isGreaterThan0, that takes in a number, myInteger, and checks if the number is greater than 0.  If the input is not an integer, return the string "invalid input, please try again".

Bonus:

10. Write a function, isEven, that takes in a number, myNumber, and returns true if the number is even.  Hint: look up the "modulo" % operator.
