Guess the Number Game 
Requirements

You may use the starter template at: http://alexrivadeneira.github.io/template

Description:
* A user enters numbers into an input box to guess a random number the computer has picked.
* If the user's input number is greater than the chosen random number, the computer tells the user to guess a lower number.
* If the user's input number is lower than the chosen random number, the computer tells the user to guess a higher number.
* If the user guesses the correct number, the computer informs them that they have won.

Requirements:
1) One < input > element with an id tag given as an attribute.
2) One < button > tag that has an onclick attribute.
3) One function that runs when the user clicks the button.  The function takes in a value from the input textbox.
4) Generate a random number between 1 and 10 using Math.random() and Math.floor()

See if you can get started with just the requirements above.  If you need more help, here are some additional hints:





































1) Add the < input > element to the page.  Make sure the type attribute is set to "text"!

2) Add an id to the input element.  Use document.getElementById to log the input element to the page.

3) Log the value of the input element.  Try adding some text to the input box in the browser and logging it to the screen.

4) Use Math.random() and Math.floor() to generate a random number every time the page loads.  Log the the value to the console.

5) Write a function higherLower that takes in an input num and checks the input against the random value.  Console.log() "higher!" if the input is lower than the random number, and "lower!" if the input is higher than the random number.  

5) Add the < button > element, and add some text to it.

6) Add an onclick event to the button element, and connect it to your function higherLower.

7) Use document.getElementById() and .value to send the < input > box's input to your higherLower function.

8) Add another < div > to the page and give it an id "message"

9) Use .innerHTML to update the message part of the page based on the function's results.
