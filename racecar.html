Racecar Project
Requirements

You may use the starter template at: http://alexrivadeneira.github.io/template

Description:
* This simple game has a racecar inside of a rectangle.
* When the user holds down any key, the car moves to the right.
* When the car hits the border of the rectangle on the opposite side, it "wraps" around to the other side of the rectangle.

Requirements:
1) One < div > element with id of "car".  The car should have the CSS position property set to "absolute" and the initial "top" and "left" positions set.  Set the car to have width "50px" and height "50px".  You need to add in a background-color as well.
2) One < div > element with id of "map" to contain the car.  The map should have the CSS position property set to "relative".  The map needs some width and height (300px by 500px should do the trick)
3) You must use the modulo ( % ) operator to calculate the position of the car and to make sure it wraps around the container rectangle.
 
See if you can get started with just the requirements above.  If you need more help, here are some additional hints:





































** Remember: At every step, you should be logging your results to the console and checking.

1) Add the map and car divs to the page (the ids are "car" and "map" respectively).  The car div goes inside the map div.  Check above for the CSS properties that each div requires.  Remember: map should be set to position: relative, and car should be set to position: absolute. 
2) Get your car div using document.getElementById 
3) Write a function "updatePosition" that uses window.getComputedStyle(car).left to get the car's current position.  Log out the current position.  You can type "updatePosition()" into the console to test the function.
4) Once you see the position logging correctly, use the split function to break the position into an array.  For example, "50px" gets split into an array like so: ["50", "px"].  The array has two strings, "50" and "px" in it.
5) Now you have to update the position.  You need to access the numerical part of the position property from the array, which is at position 0.  If you assigned your array the variable name "positionArr", for example, you could access "50" at positionArr[0].
6) Since your positionArr[0] is a string, if you want to do math on it, you'll have to convert it to an integer first.  Use parseInt() to convert it.
7) Once you've converted your position to an integer, add 10 or 20 to it.
8) Inside your function, update the car div's position to the new position using car.style.position.left.  Remember, this gets set to a string with "px" at the end of it!
9) Add an event handler to your  tag, onkeypress which runs the function you wrote to update the car's position!
